# BuscadorTexto

Aplicación de escritorio desarrollada en Java con interfaz gráfica que permite al usuario seleccionar un fichero de texto (.txt) y realizar búsquedas de palabras o frases en su contenido.

## Requisitos del sistema
- Sistema operativo: Windows 7 o superior
- Java Runtime Environment (JRE) 8 o superior
Si no tienes Java instalado, puedes descargarlo desde la página oficial:
https://www.java.com/es/download/


## Instalación

### Opción 1 — Instalador de Windows
La forma más sencilla de instalar la aplicación es mediante el instalador para Windows incluido en la sección Releases del repositorio.
1. Descarga el archivo `InstaladorBuscadorTexto.exe` desde Releases
2. Ejecuta el instalador y sigue los pasos del asistente
3. Una vez finalizada la instalación, se creará un acceso directo en el Escritorio
4. Abre la aplicación desde dicho acceso directo

### Opción 2 — JAR ejecutable
Si prefieres ejecutar la aplicación directamente sin instalarla:
1. Descarga el archivo `Fuentes_Tarea_U07.jar` y la carpeta `lib` desde Releases
2. Coloca ambos en el mismo directorio
3. Ejecuta el JAR con doble clic, o bien desde la terminal.
